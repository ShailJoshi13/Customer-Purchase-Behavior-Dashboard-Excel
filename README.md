# Excel Customer Purchase Dashboard

**Interactive Excel dashboard** built to analyze customer purchase behavior and shipping performance using real-world retail data. The dashboard demonstrates Excel-based Business Intelligence (BI) skills including data cleaning, PivotTables, charts, slicers, and KPI design.

---

## Project Summary
This project converts raw transaction data into actionable insights by using Excel PivotTables, calculated fields, and interactive visuals. Key analyses include revenue by category, seasonality, payment method performance, geographic revenue distribution, color preferences, and purchase frequency.

---

## Files included
- `Excel Customer Purchase Dashboard.xlsx` — the interactive dashboard file (PivotTables, charts, slicers, KPI cards)
- `purchase_shipping_data.xlsx` — sample dataset used to build the dashboard (if included)
- `README.md` — this file

---

## Key Features
- Data cleaning (Remove Duplicates, Data Validation, Text to Columns)
- Dynamic PivotTables & Calculated Fields (Total Revenue, Average Rating, Customer Count)
- Interactive Slicers for Season, Category, Location, Gender, Size, and Color
- Multiple chart types: Column, Pie, Combo, Filled Map, Clustered Column, Radar
- KPI cards for quick overview of Total Revenue, Average Age, Avg Rating, Repeat Purchases, Customer Count
- Map visualization for geographic revenue analysis

---

## Tools & Skills
- Microsoft Excel (Pivot Tables, Charts, Slicers, Calculated Fields)
- Data cleaning & formatting
- Dashboard layout and design
- Business insight extraction & reporting

---

## Key Insights
- Clothing generates the highest revenue.
- Frequent buyers (weekly/monthly) contribute a large share of total sales.
- Cash and Credit Card payment methods show higher customer satisfaction (average rating).
- Revenue distribution is relatively stable across seasons; certain states show stronger revenue contribution.

---

## How to preview
- Open `Excel Customer Purchase Dashboard.xlsx` in Microsoft Excel (desktop version recommended for full slicer & map functionality).
- Use slicers on the dashboard sheet to interactively filter data across visuals.
